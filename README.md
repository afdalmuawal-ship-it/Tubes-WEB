# LOSTLY

## Deskripsi
LOSTLY adalah aplikasi berbasis web yang digunakan untuk membantu proses pelaporan, pencarian, dan pengelolaan barang hilang.

## Fitur
- Login pengguna
- Pelaporan barang hilang
- Pelaporan barang temuan
- Pencarian barang

## Teknologi
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## Cara Menjalankan
1. Buat proyek ke folder `htdocs`.
2. Buat Database dan Tabel serta tambahkan data awal untuk web Lostly.
3. Jalankan Apache dan MySQL melalui XAMPP.
4. Buka aplikasi melalui browser.
